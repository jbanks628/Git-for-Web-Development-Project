    1. What is Semantic HTML? 
  Semantic HTML clearly gives meaning to a page in a readable way.

    2. What is HTML used for? 
  HTML is used for organizing basic web page content.

    3. What is an attribute and where do we put it? 
    Attributes are elements used to include info in an element that we don't want to render on the page.
    We put them in between the anchor tags.

    4. What is the h1 tag used for? How many times should I use it on a page?
    The h1 tag is used for the main heading with the largest font. We only used it once on a page.

    5. Name two tags that have required attributes:
    The anchor <a> and image <img> tags both have required attributes.

    6. What do we put in the head of our HTML document? 
    The title of our page/content will go in the heading of our HTML document.

    7. What is an id? 
    Ids are attributes with titles that can only appear on a single element.

    8. What elements can I add an id to?
    Any HTML element can be added to the id attribute.

    9. How many times can I use the same id on a page? 
    You can only use it once.

    10. What is a class? 
    Classes are attributes with titles that can be used on multiple elements.

    11. What elements can I add a class to? 
    All elements can be added to a class.

    12. How many times can I use the same class on a page? 
    It can be added however many times.

    13. How do I get my link to open in a new tab?
    Target=_blank

    14. What is the alt attribute in the image tag used for? 
    It  is an alternate text for an image if it can't be displayed.

    15. How do I reference an id?
    You can reference an id by a hashtag #.

    16. What is the difference between a section and a div?
    A section is semantic, and div is non-semantic.

    17. What is CSS used for? 
    CSS is used to style the layout/presentation of the webpage.

    18. How do we select an element? Example - every h2 on the page.
    Select them by surrounding in angle brackets. <element>

    19. What is the difference between a class and an id? - Give me an example of when I might use each one.
    An id uses a unique identifier for an element, while a class can use multiple. When styling 1 element on the page, use id. 
    When wanting to apply a style to multiple elements use class.

    20. How do we select classes in CSS?
    We select classes by using a "." before the element.

    21. How do we select a p element with a single class of “human””?
    p.human {"";}

    22. What is a parent child selector? When would this be useful? 
    A parent selector selects all elements that are a child of a certain element. The child is nested in the parent selector.

    23. How do you select all links within a div with the class of sidebar?
    div .sidebar{"";}

    24. What is a pseudo selector?
    It selects elements in a specific state. 

    25. What do we use to change the spacing between lines?
    line-height:

    26. What do we use to change the spacing between letters?
    letter-spacing:

    27. What do we use to to change everything to CAPITALS? lowercase? Capitalize?
    text-transform: capitalize;
    "": lowercase;
    "": uppercase;

    28. How do I add a 1px border around my div that is dotted and black?
    div {

    }

    29. How do I select everything on the page?

    30. How do I write a comment in CSS?

    Block element starts a new line, while inline stays on the same line.

    2. What happens when an element is positioned absolutely? 
    It cannot move and is fixed in that position.

    3. How do I make an element take up only the amount of space it needs but also have the ability to give it a width? 
    float in CSS

    Block by default: div, header, li
    Inline: span, button

    5. In your own words, explain the box model. What is the "fix" for the box model, in other words, how do we make all elements respect the width we've given them?
    By specifying the border, margin, content, and padding, all the contents will respect those specifications.
